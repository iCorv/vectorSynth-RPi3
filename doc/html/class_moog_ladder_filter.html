Detailed Description

Applies a 4-stage low-pass filter to a sample.

This class applies a 4-stage low-pass filter to a sample. The filter is inspired by Moog Ladder Filter with Oberheim Xpander variations as described in: Designing Software Synthesizer Plug-Ins in C++, Page 404, 7.6 Moog Ladder Filter Model
