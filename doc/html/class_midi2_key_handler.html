Detailed Description

Manages incoming midi messages and handels all keys (voices).

Messages from the midiMan-class are managed and changes in parameters are set in all keys. This class holds an array of keys and keeps track on pressed/released keys. Volume LFO and Cut-off LFO are applied on this level.
